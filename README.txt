# Juke

Twitch plays Spotify.

Use: Make the sure the necessary dependencies are installed.  In the server and client folders, there are
package.json files that specify what is needed. Run:
 $ npm install
 in the server and client folders.

 The module 'lodash' is also needed for the server folder.  The module 'nodemon' is needed for both as well.

 After all the installations, the run the server.js file in both the server and client folders with
 $nodemon server.js

 In a browser, connect to localhost:9000 to begin using the app.
