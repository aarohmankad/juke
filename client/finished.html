Getting Started With Juke


"Help" Command

You can be reminded of the formatting guidelines at anytime by using the help command:

!juke help

Adding Songs to the Playlist


"Add" Command

Songs are added to a playlist called "Jukebox" in the Spotify client. Music can be added to the playlist using the following syntax:

!juke /add [Song Title] - [Song Artist]
